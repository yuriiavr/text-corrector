# AI Fixer: Системний процесор тексту низького рівня

Високопродуктивна утиліта для Windows на базі **Python 3.10+** та **PyQt6**, що використовує моделі **Google Gemini 2.0/3.1** для трансформації тексту в реальному часі за допомогою глобальних гарячих клавіш.

## 🛠 Технічна архітектура
- **Перехоплення вводу:** Використання бібліотеки `keyboard` для глобальних слухачів та `ctypes` (Win32 API) для низькорівневої ін'єкції подій клавіатури.
- **Керування буфером:** Реалізація `pyperclip` для I/O операцій з автоматичним очищенням буфера для запобігання стану гонитви (race conditions).
- **Ін'єкція процесів:** Використання `user32.keybd_event` для обходу втручання драйверів високого рівня (наприклад, конфліктів мультимедійних клавіш гучності).
- **Асинхронне виконання:** Усі API-запити обробляються в окремих потоках `threading.Thread` (daemon mode), що гарантує стабільність UI під час мережевих затримок.
- **Персистентність:** Керування налаштуваннями через `QSettings` (реєстр Windows) та інтеграція автозапуску через `winreg` (HKCU Run-key).

## 🚀 Основні характеристики
* **Zero-UI Workflow:** Обробка тексту безпосередньо в будь-якому активному вікні без перемикання контексту.
* **Низькорівнева емуляція:** Ін'єкція віртуальних кодів клавіш (`VK_CONTROL`, `VK_C`, `VK_V`) забезпечує сумісність із ізольованими програмами.
* **Prompt Engineering:** Спеціалізовані системні інструкції для трьох режимів: `Fix` (граматика), `Pro` (діловий стиль) та `Translate` (переклад).
* **Динамічний вибір моделей:** Підтримка актуальних моделей `gemini-2.5-flash`, `gemini-2.5-flash-lite` та `gemini-3.1-flash-lite-preview`.
